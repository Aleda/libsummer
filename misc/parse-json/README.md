# parse-json
=======================================================
### What Is Parse Json?
Parse-json 基于开源cJSON库实现的一个简单的转换json
数据的工具.
LISENSE继承cJSON.

### How To Use It?

    step 1:
    make
    step 2:
    ./parse-json;
    cat input.json | ./parse-json hometown/zipCode[1]
    or
    bash test.sh
